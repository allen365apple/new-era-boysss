# 新世紀直男戰士 — 改風格小抄

> 想換顏色、字體、底色?**全部都在一個檔案裡改。**
> 檔案位置:`src/styles/global.css`，最上面的 `:root { ... }` 區塊。
> 改完存檔 → push 到 GitHub → Cloudflare 自動部署 → 全站一起變。

---

## 一、顏色

| 變數 | 現在的值 | 控制什麼 |
|------|----------|----------|
| `--accent` | `#2d6dc3`（藍） | **主色**:文章標題、章節標題、連結、集數/議題標籤、目錄高亮 |
| `--accent-strong` | `#0066ff` | 更鮮豔的強調藍（備用，目前少用） |
| `--accent-dark` | `#1e4f96` | 滑鼠移到連結上時的深色 |
| `--accent-light` | `#e8f0fa` | 淺藍底:議題標籤背景、目錄高亮的底色 |
| `--accent-gold` | `#fad13b` | 金色點綴（備用） |
| `--surface` | `#fbf7f1` | **文章底色**（暖米白）。想要純白就改 `#ffffff` |
| `--surface-soft` | `#f4eee4` | 區塊底色:文末收聽區、引言框 |
| `--border` | `#ddd5c8` | 分隔線（虛線）的顏色 |
| `--text-primary` | `#1f2430` | 主要內文文字（最深） |
| `--text-secondary` | `#4a4f5c` | 次要文字（中等灰） |
| `--text-muted` | `#9a958c` | 提示性文字、日期（最淺） |

### 範例

```css
/* 想把整站改成綠色:只改這一行 */
--accent: #1f9d5c;

/* 想把暖米色底改成純白:只改這一行 */
--surface: #ffffff;
```

改 `--accent` 一行，首頁標題、文章標題、所有標籤、連結、目錄高亮會**同時**變色。

---

## 二、字體

| 變數 | 控制什麼 |
|------|----------|
| `--font-sans` | 全站字體。目前是思源黑體 `"Noto Sans TC"` |

換字體只改這一行。第一個字型是主要字型，後面是備用（找不到時往後用）。

---

## 三、版面尺寸

| 變數 | 現在的值 | 控制什麼 |
|------|----------|----------|
| `--content-width` | `680px` | 文章內文的寬度（想要更寬的閱讀區就調大） |
| `--radius` | `10px` | 一般圓角（按鈕、輸入框） |
| `--radius-lg` | `16px` | 大圓角（卡片、區塊） |

## 找色碼的小工具

不知道想要的顏色色碼?Google 搜尋「**color picker**」會跳出取色盤，
選好顏色複製那個 `#xxxxxx`，貼到對應變數即可。

---

## 重要原則

- **不要去各個元件檔案裡改顏色**，那會改得零零落落。永遠改 `global.css` 的 `:root` 變數。
- 改完想看效果:本機 `npm run dev` 開 `localhost:4321`，或 push 後看線上網站。
- 不確定改哪個 → 直接問 Claude，貼上你想要的效果。
